SELETORES BOX MODEL

SELETORES

Seletor Universal

* -> universal (seleciona tudo)

Modo de uso: * {propriedade: valor} -> *{font-sise: 22px} // elemento * {prop: valor} -> p* = para todo elemento p



Seletor Tipo

Mode de uso: elemento {propriedade: valor} -> h3{cor: red}



Seletores Agrupados

, -> separar grupo de TAGs com vírgula

Modo de uso: elemento, elemento, elemento {prop: valor} -> h1, h2, p {color: blue}



Seletor Descendente

Defini a configuração para um elemento (tag) que está dentro de uma certa sequência de outros elementos (hierarquia).

Modo de Uso: tag_1 tag_2(dentro da tag_1) {prop: valor} -> div p {font-family: Arial...}



Classes e estilos individuais

Uma classe pode ser associada a um elemento (tag). Com isso, podemos aplicar diferentes formatos no mesmo elemento, bastando chamar as respectivas classes.

Modo de uso: elemento . nome-da-classe {prop: valor} -> p.cor_rosa {color: pink}



Seletor Classe (genérico)

. -> usado antes do nome da classe

Modo de uso: . nome-da-classe {prop: valor} -> .subtitulo {text-align: center}



Seletor ID

# -> usado antes do nome do ID

Modo de uso: # nome-da-classe {prop: valor} -> #conteudo {text-align: justify}
